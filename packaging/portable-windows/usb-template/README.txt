Hermes 助手 · 便携版
==========================

只需要做这 4 件事:

  1. 把这个 U 盘插上电脑
  2. 双击 U 盘里的 🚀 "启动 Hermes 助手"
  3. 第一次会等大约 2 分钟,之后浏览器会自动打开聊天界面
  4. 用完后,在屏幕右下角找到 🟡 小图标 → 右键 → "安全退出并弹出 U 盘"

需要 Windows 10 (21H2 或更新) 或 Windows 11。

数据全部保存在 U 盘的 data\ 文件夹里 ——
换一台电脑插上,聊天记录、设置都跟着走。

遇到问题?查看同目录下的 "🆘 问题排查.txt"。
