问题排查 · Hermes 助手便携版
================================

【1】双击没反应,或者只闪了一下

原因往往是 Windows 子系统(WSL)没有开启。
解决:
  1. 打开"设置" → "应用" → "可选功能" → "更多 Windows 功能"
  2. 勾选 "适用于 Linux 的 Windows 子系统" 和 "虚拟机平台"
  3. 重启电脑后再次双击启动器

如果你是公司发的电脑,可能被 IT 锁定了 WSL。请联系 IT 解禁,或换一台个人电脑。

【2】首次启动很慢

第一次插到一台新电脑上需要约 2 分钟来注册系统。
之后插到同一台电脑就只需要几秒钟。

【3】杀毒软件弹窗 / SmartScreen 警告 "未知的发布者"

发行版本应该已经带了代码签名 —— 如果你看到警告,说明你拿到的可能不是正式发布版。
请联系 U 盘提供方,或在 SmartScreen 上点 "更多信息" → "仍要运行"。

【4】浏览器没自动打开

打开 U 盘里的 data\cache\dashboard-token.txt,找到 url=...,
把这个地址复制到浏览器地址栏即可。

【5】聊天界面打开了但报 API key 错误

第一次需要在界面里 "设置" 标签页填入你的 API 密钥(OpenAI / Anthropic / 其他)。
密钥保存在 U 盘里,加密存储。

【6】U 盘拔不出来 / "正在使用中"

正确做法:
  右下角 🟡 小图标 → 右键 → "安全退出并弹出 U 盘"
而不是直接拔。
直接拔可能丢失最近的聊天记录(SQLite WAL 还没写回 U 盘)。

【7】不想再用了 / 想清理这台电脑上的痕迹

  右下角 🟡 小图标 → 右键 → "高级" → "退出时彻底卸载"
然后重启 Hermes 助手并安全退出 —— 这台电脑上的注册项会被完全清除。

【8】查看运行日志

  右下角 🟡 小图标 → 右键 → "查看日志"
会打开 U 盘的 data\logs 文件夹。
把最新的 .log 文件发给 U 盘提供方,有助于排查问题。
